# 仁迈

仁迈是一款面向 Windows 桌面环境的聊天记录整理与关系分析软件。软件以用户主动导入的聊天记录文件为处理对象，对文本内容进行解析、归类、统计和结构化输出，形成联系人关系评估结果、行动建议与礼物建议，适用于个人社交信息整理与辅助分析场景。

## 软件概述

- 软件名称：仁迈
- 版本号：V1.0.0
- 软件形态：Windows 桌面应用软件
- 开发框架：Flutter
- 运行方式：本地安装、本地使用
- 处理对象：用户主动提供的聊天记录文件

## 主要功能

当前版本主要提供以下功能：

1. 支持导入 `.txt`、`.html`、`.htm`、`.zip` 等聊天记录文件。
2. 支持导入包含聊天记录文件的文件夹。
3. 对导入内容进行来源识别、联系人识别、时间识别和消息整理。
4. 对消息内容进行去重、标准化和联系人聚合。
5. 生成联系人关系排序、活跃度评估、风险提示、行动建议和礼物建议。
6. 支持由用户手动启用 AI 增强分析，对本地报告进行补充说明。

## 适用范围

本软件适用于已经具备可阅读聊天记录文件的使用场景。建议导入的数据形式包括：

- 普通文本格式聊天记录文件
- 网页格式聊天记录文件
- 由聊天记录文件组成的压缩包
- 包含上述文件的文件夹

当前版本不以直接解析微信、QQ 等软件内部数据库为主要能力，也不以截图、录屏、缓存文件作为标准输入源。

## 基本使用流程

1. 准备聊天记录文件或文件夹。
2. 启动仁迈软件并进入导入界面。
3. 选择待导入的文件或目录。
4. 执行导入与解析。
5. 在综合报告、联系人洞察等页面查看分析结果。
6. 按需在 AI 设置页面配置模型接口并手动触发 AI 增强。

## 功能页面说明

- 仪表盘：用于展示导入概况、数据摘要和待处理事项。
- 导入向导：用于选择文件、目录并执行导入。
- 综合报告：用于展示关系排序、风险提示、行动建议和礼物建议。
- 联系人洞察：用于查看单个联系人的分析明细与证据片段。
- AI 设置：用于配置外部模型接口信息及 AI 增强开关。

## 隐私与安全说明

1. 本软件不提供在线即时聊天功能。
2. 聊天记录在默认情况下仅在本地环境中整理、保存和分析。
3. 只有在用户主动启用 AI 增强功能时，软件才会调用用户自行配置的外部模型接口。
4. 接口密钥等敏感配置采用本地安全存储方式保存。

## 工程目录

```text
lib/
├── config/                 # 路由、主题、常量定义
├── models/                 # 数据模型定义
├── providers/              # 状态管理与业务协调
├── screens/                # 界面页面
├── services/               # 导入、分析、存储、AI 调用等服务
└── utils/                  # 通用工具与辅助方法
```
